Learning Node.js

为什么要学Node.js

  • 首先,Node.js是用google的V8引擎写的,所以,node可以用js的语法,省去一些其它的学习成本,比如php(不能学世界上最好的语言真的好可惜!)逃).
  • 其次,Node.js实现后端是靠一些api,就是封装好的函数实现的,所以,需要了解一些基本的API,和后端的方法

  • 最后,也是最重要的,Node.js上面有很多别人写好的模块,很好用的,很火的,比如Express,比如jade(现在改名了.是个快速生成HTML模板的模块.),但是要是把精力都浪费在追逐这些东西上,那可没头.无论模块怎么写,都是要遵循Node.js的核心,比如function(err, callback){}.所以有一些核心API,一定要知道,剩下的就到时候现查表就是了,怎么用都写的很明白.

我摸索出来的学习路线是:

  1. 熟悉基本的API,达到能用的程度先.
  2. 着重看核心的原理和核心api
  3. 做做小项目,熟悉热门模块.
